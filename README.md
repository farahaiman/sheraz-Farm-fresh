# 🌾 Sheraz Farm Fresh

A modern and responsive agriculture business website designed for showcasing premium wheat and rice products directly from farmers.

## 🔗 Live Demo

👉 https://farahaiman.github.io/sheraz-Farm-fresh/

## 📖 Project Overview

Sheraz Farm Fresh is a simple and user-friendly website created to promote fresh agricultural products. The website highlights wheat and rice products, company information, and contact details in a clean and responsive layout.

## ✨ Features

- Responsive Design
- Modern User Interface
- Hero Section with Call-to-Action
- Product Showcase Section
- About Us Section
- Contact Information
- Mobile-Friendly Layout
- Easy Navigation

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages (Deployment)

## 📂 Project Structure

```
Sheraz-Farm-fresh/
│
├── index.html

└── README.md
```

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/farahaiman/sheraz-Farm-fresh.git
```

2. Open the project folder

```bash
cd sheraz-Farm-fresh
```

3. Open `index.html` in your browser.
